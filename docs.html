Getting Started

Live in three steps.

01
Connect your store
Install the Dropt app on Shopify. Tagged orders flow into Dropt automatically — no replatforming, no code on your storefront.
02
Connect your OMS/WMS
Link your order and warehouse systems so store inventory, stock reservations and delivery status stay in sync both ways.
03
Configure zones & go live
Set delivery zones and SLAs per store from the dashboard, surface live ETAs at checkout, and start fulfilling same-day.
API Overview

One API for the whole delivery lifecycle.

Serviceability / ETA
Check delivery availability and a live ETA for any pincode before checkout, ranked to the fastest serviceable store.
Orders
Create delivery orders and track them through accept, pack, pickup and delivered — each with timestamps and status.
Webhooks
Receive status callbacks for every lifecycle event so your systems update in near real time without polling.
Inventory Sync
Keep per-store stock and reservations accurate so serviceability only ever offers what is actually available.
Tracking
Surface a live tracking link with rider location, ETA and delivery OTP on your own domain after checkout.

Authentication

Authenticate with a Bearer token.

Each brand generates an API key from the dashboard under the API & Webhooks page. Pass it as a Bearer token on every request. Keys are scoped per brand, so your data stays fully isolated.
